04 — One word, every disguise

The constellation of a single swear

Take one root and pull every banned string that decodes to it across all 12 states. The clean spelling sits at the center; each ring outward is one more edit away. The center is caught widely — the exotic spellings at the rim were each caught by a single state that happened to imagine them.

Source: 12 state DMV vanity-plate rejection lists, public-records requests via governmentattic.org / MuckRock, c. 2012–13.  Method: a string counts as a variant of the root if its skeleton — leetspeak decoded, PH→F · CK→K · Q→K · Z→S folded, vowels dropped — equals the root's; rings = Levenshtein distance from the clean spelling.  Limit: the skeleton key is approximate (the same looseness we rejected in an earlier draft); it errs toward over-grouping, so treat rim membership as suggestive, not exact.

05 — Do they even agree

The moral-distance matrix

For all 66 state pairs, overlap = shared strings ÷ combined strings. If America had a common standard, this grid would glow. It doesn't. The average pair shares — of its banned plates. The grid is almost entirely cold — agreement is the rare bright cell, not the rule.

Darkest = identical lists (only the diagonal). The warmest off-diagonal pair, NY·IA, overlaps just 7.8%. Most pairs are near zero. Twelve governments policing "offensive," almost none of it the same words.

Source: the same 12 rejection lists (governmentattic.org / MuckRock, c. 2012–13).  Method: Jaccard index — shared strings ÷ combined strings — over all 66 state pairs.  Limit: overlap is measured on exact strings, so FUCK and FUC count as zero agreement even though both ban the same word. This inflates the disagreement — the true conceptual overlap is higher than 1.95%. The finding survives anyway: even the warmest pair shares under a tenth of its list.

06 — The number language

What the digits are really saying

Plates hide meaning in numbers. Some codes are real — the digits genuinely carry the forbidden meaning. Others are a Rorschach test: numbers that look sinister but, read in context, are filler or leetspeak. We split them, because the difference is the discipline.

07 — The outlier

New Mexico rejects cleverness

The smallest list in the set — 77 strings — and the strangest. Where other states reject meaning, New Mexico's list reads like a rebus puzzle book: plates rejected for being too cryptic to parse, not too crude. Tap a tile to decode it.

It isn't that NM bans less profanity — it sits mid-pack there (5.2%). It's that its rejections are a different kind of object: wordplay and leetspeak riddles, not slurs.

Source: New Mexico's rejection list (governmentattic.org / MuckRock, c. 2012–13), n=77 — the smallest in the set.  Method: profanity share = fraction of a state's strings whose skeleton matches a known profane root; decodings are hand-read.  Limit: "cleverness" is a qualitative read, not a metric, and n=77 is small — this is an outlier portrait, not a statistical claim. Decodings are our best interpretation; some plates are genuinely ambiguous.

How we did this

The data

Two real corpora. (1) Twelve states' vanity-plate rejection lists — NY, OH, OK, KS, NJ, UT, IA, CO, NM, GA, WI, CA — obtained through public-records requests and mirrored on governmentattic.org and MuckRock, dated roughly 2012–13. (2) California's published approve/deny review applications (Veltman / CA DMV, 2015–16), which carry an actual verdict. Vintage matters: these are decade-old snapshots, not today's lists — read every finding as "as of ~2013."

01–03 · The evasion simulation

For a plate and a state we check that state's actual list three ways: literal membership, edit-distance-1, and a phonetic-key match — no hand-tuned probabilities, a state can only catch what it wrote down. The 400-submission sim draws a random state each trial and asks whether it catches the plate; the slip rate is the share approved. Limit: detection sees only the published list, so slip rates are an upper bound on leniency — a clerk may reject on judgment what never made the list.

04 · The constellation

Every banned string whose decoded skeleton (leetspeak resolved, PH→F · CK→K · Q→K · Z→S, vowels dropped) equals the root, placed by Levenshtein distance from the clean spelling. Limit: the skeleton key is deliberately loose and over-groups; rim membership is suggestive, not proof a plate "means" the root.

05 · The moral-distance matrix

Jaccard index — shared ÷ combined strings — across all 66 state pairs. Limit: measured on exact strings, so FUCK vs FUC reads as zero overlap. This inflates the disagreement; true conceptual overlap is higher than the 1.95% mean. The conclusion holds regardless — the warmest pair still shares under 8%.

06 · The number language

Plates counted by the numeric codes they contain, then sorted into verified (the digits dominantly carry the forbidden meaning in this corpus — 69, 420, 666, 187) and mirage (read in context they're filler or leetspeak — 88, 14, 311). Limit: intent can't be proven from a plate; no single plate is certified coded, and the hate-code reading is explicitly not supported here.
